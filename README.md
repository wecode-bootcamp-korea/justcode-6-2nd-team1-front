### 일일일차(一日一茶_팀의 **`공차`** 모바일웹 Clone 코딩 프로젝트입니다.

---

## **프로젝트 소개**

```
카페 공차 모바일 웹 사이트를 클론하여 전반적인 UI를 구성하였고 어플을 참고하여 스마트 오더 기능을 추가로 구현했습니다.
로그인 시 현재 위치 정보를 받아서 가장 가까운 공차 매장을 보여주고 선택할 수 있게 했습니다. 이후 스마트오더 기능을 해당 매장에 음료 주문하는 기능입니다.
```
<br>
